# deliberate Brand Assets

Use these logo marks for the app identity and splash/loading states.

## Files

- `deliberate-mark-transparent.png`
  - Black sumi-e / shodo argument-map mark with transparent background.
  - Best for overlaying on the paper UI, splash screen, favicon source, loading states, and subtle background marks.

- `deliberate-mark-paper.png`
  - Same black mark on an off-white paper texture.
  - Best for Devpost screenshots, pitch materials, or quick standalone display.

## Usage Notes

- Do not add the wordmark directly into the image.
- In the app UI, render the text `deliberate` separately with minimalist typography.
- Keep the transparent mark subtle when used as background decoration.
- Avoid recoloring unless the whole visual system changes.
